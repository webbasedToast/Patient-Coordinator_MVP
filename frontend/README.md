# Patient Coordinator - Frontend
Webfrontend des Patient Coordinator MVP.
Das Frontend stellt ein User Interface zur Verwaltung von Patiententransporten zur Verfügung.

## Technologies
- React / TypeScript
- Vite
- Material UI
- SCSS

## Voraussetzungen
- [npm und Node.js](https://nodejs.org/en/download) müssen installiert sein

## Installation

1. In das Root-Verzeichnis des Projekts wechseln und Abhängigkeiten und Packages installieren:
    ````bash
    cd Patient-Coordinator_MVP\frontend
    npm install
    ````

## Ausführen

Der Entwicklungsserver wird mittels `npm` über die Commandline gestartet
   ```bash
   npm run dev
   ```
Die Adresse unter der das Frontend erreicht werden kann, ist anschließend der Commandline zu entnehmen.
